## ℹ️ General Info

- [Backend](./backend) — To work properly, fill in the **`.env`** file. Use the **`.env.example`** file as an example.

## Start

1. Clone repository
2. Fill ENVs
4. **`cd frontend && npm i && npm run dev`** then **`cd backend && npm run i && npm run dev`**
